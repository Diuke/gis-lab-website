Intercomparison Results

Output and statistics of the intercomparison step.

  • Output:

    In the intercomparison step we generated three rasters: the GHS-POP and WorldPOP aligned and clipped to the extent of our tiles, and a raster with the difference of them — GHS-POP subtracted WorldPOP.
    All three layers can be seen on the WebGIS.

  • Statistics:

    We calculated univariate statistics for the differences raster using GRASS r.univar function and correlation values between GHS-POP and WorldPOP rasters using GRASS r.covar function.
    Correlation values were calculated for the complete raster, and for each one of the tiles, i.e. correlation of both rasters clipped in single tile.

The variance and standard deviation are also big, probably due to those values.

    We can consider 2 kind of tiles: Tiles containing the small islands that have very few amount of data (always less than 4000), and tiles containing the mainland and big islands (Falkland Islands) that have a lot of data available. The correlation in small islands tiles is very variable, ranging from 0.230698 to 0.873545. Values of 1 are left aside because the islands in those tiles are inhabited and both datasets properly label all pixels as 0.
    The variability in correlation may be explained by the little amount of data, making it very succeptible to outliers.

    On the other side, the mainland and big islands tiles are generally more robust, ranging from 0.516338 to 0.983315, always above 0.5.
    The low value 0.516338 corresponds to half of the Falkland islands, and is the tile with less data of the mainland and big islands tiles with 19973 pixels. The second lower value is 0.6931 and corresponds to the mainly inhabited area of Magallanes province in Chile and half of the Argentinian Patagonia. This places have very low population density and, specially the Magallanes province, is very geographically complex. Outliers in the calculation of the values could lead to the low correlation value.
    Aside from those values, all correlation values in mainland and big island tiles exceed 0.8.

Validation Results

Output and statistics of the validation step.

  • Output:

    In the validation step we generated three rasters: the GHS-POP and WorldPOP classified to populated (value = 2) or not populated (value = 1) using the rasters previously processed in the intercomparison, and a raster calculating the differences between the classification rasters, with the following values:
    0 = GHS not populated but WorldPOP populated.
    1 = GHS and WorldPOP either both populated or both unpopulated.
    2 = GHS populated and WorldPOP not populated.

    Also, we generated 600 sampling points with the QGIS AcATaMa plugin that were classified using satellite imagery from Bing Satellite and Google Satellite and used as the reference for the validation.

    Finally, we generated accuracy statistics for both GHS-POP and WorldPOP classified rasters and the error matrix (also called confusion matrix) using a python script.
    The accuracy indices measure:
    Overall accuracy (OA): Proportion of correctly classified pixels in total number of pixels.
    User’s accuracy (UA): Shows how often a user of thematic raster can expect to find the class on the ground.
    Producer’s accuracy (PA): Probability that the class present on the ground is also captured by classification in the thematic raster.

    All layers can be seen on the WebGIS.

  • Sampling:

    The sampling was performed using the QGIS plugin AcATaMa. It generated 600 points using the categories from the raster with the differences between the classified GHS-POP and WorldPOP.
    The distribution of the samples is as follows:
    - 33% of points where GHS-POP not populated, but WorldPOP populated (category 0)
    - 34% of points where GHS-POP and WorldPOP either both populated or unpopulated (category 1)
    - 33% of points where WorldPOP not populated, but GHS-POP populated (category 2)

    We did 3 random samplings of 200 points, one for each member of the group, and merged the results into one single vector layer. In each individual sampling the distance between samples is at least 0.08 degrees because of to the raster resolution, but in the complete sampling points can be closer.

    This classified sampled points are the reference data for the validation, since the classification was done manually using high-resolution satellite images.

    Results Interpretation

    After performing the validation and accuracy analysis with the reference samples we obtained from high-resultion satellite imagery, we concluded that that WorldPOP map has a higher accuracy in overall accuracy and in both the user accuracy and producer accuracy for both classes with respect to GHS-POP.

    In both maps the accuracy for not populated is much higher than the accuracy for populated values. Looking at the classified rasters in the WebGIS we can see that WorldPOP has much more area classified as populated than GHS-POP.

    The error matrices show in the main diagonal the elements that were correctly classified with respect to the category, and the off-diagonal elements show the elements that were wrongly classified. In both error matrices we can see that on the populated class more than double of the correctly classified points were missclassified, with a slight improvement in WorldPOP. This explains the low accuracy for the populated category.
    The higher accuracy in populated class for WorldPOP can be explained by the fact that WorldPOP have more area classified as populated than GHS-POP.
